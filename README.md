# sveltesse

## Stack

- SvelteKit
- Svelte Remote functions
- Tailwind CSS
- shadcn-svelte
- Lucide Icons
- Better-Auth
- Drizzle ORM

**features:**

- SEO-ready
- Error handling
- Useful stores
- Common libraries
- Utilities
- Strict TypeScript enforcement
- Environment-based config
- Cloudflare Workers ready
